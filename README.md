# Union Living - Real Estate Landing Page

A modern, responsive real estate landing page built with React and Vite, designed to showcase student housing properties across top Indian cities.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Hero Section**: Full-screen video background with overlay
- **Navigation Bar**: Sticky navigation with announcement bar
- **City Cards**: Interactive cards showcasing properties in different cities with color-coded themes
- **Coming Soon Badge**: Special ribbon badge for upcoming locations
- **Smooth Animations**: Subtle hover effects and transitions
- **Modern UI**: Clean and professional design with gradient accents

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd union-living-assign
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## 📦 Build

To create a production build:

```bash
npm run build
```

The build output will be in the `dist` directory.
